      - bundle install

    • Database creation

      -rake db:create
-rake db:migrate

      simple tutorial here : http://blog.mohitkanwal.com/blog/2013/04/10/deploying-rails-on-nginx-and-thin/

      but you can disable thin for development purpose on the Gemfile to use webrick.
